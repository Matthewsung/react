loader의 preset은 plugin들의 집합으로 target으로 각각 지정도 가능 - module이나 rules에서 지원하는 plugin외에 적용하는 plugin을 지원 9) require / import : 동일하게 파일을 가져오고 내보낼때 사용 / 둘이 다르긴하지만 호환이 된다(react에 들어가보면 import를 require로 바꾸는 문법이 있어서 가능) const React = require ('react') === import React from 'react' module.exports = XXX === export default XXX 10) 배열 문법 new Set (배열) ->중복을 제거하는 방법 / new Set(배열).size ->몇 자리수인지 확인(legnth와 동일) 배열.join('') ->배열을 문자열로 변경 11) 최적화 PureComponent (class) / memo(Hooks) 를 사용하거나 shouldComponentUpdate로 각자 조건을 만들어주면 변하는 부분만 랜더링 됨 12) Lifecycle : component가 나타났다 사라졌다 -
